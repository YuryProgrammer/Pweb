Questão 4 - RESPOSTAS:

A) Title: Serve para dar o título da página,por exemplo, Google.com.

B) DIV: Serve para criarmos um bloco moldável em que podemos dar cor, dar borda, links e adicionar as funcionalidades do HTML e CSS.

C) ul e ol: ul, serve para criar listas desordenadas e ol para criar listas ordenadas.

D) b: Serve para dar um espaçamento entre os códigos ou para parar um código em específico, quebando a linha.

E) a: Serve para deixar as palavras em negrito.

F) img: Serve para colocar imagens na página.
