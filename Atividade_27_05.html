RESPOSTAS

1- A)

a) .caixa, é a parte/div do código que está sendo alterado;.
- Width é a lagura, que agora equivale a 300px; - Height é a altura, que agora equivale a 150px; - Border é a borda que está sendo adicionada nesse código que equivale a 2px no estilo solid com uma “fonte” navy;
- Padding, é o espaçamento que o texto dentro dessa div tem da borda, que é 20px;
- Margin, é o espaçamento que as divs tem entre elas, nesse caso é 30px de distância;
- Color, é a cor que o texto dentro dessa div tem, ou seja, darkblue;
- Text-Align, é onde o texto está na div, aqui ele está localizado no centro.

1- B)

b) .aviso é a parte/div do código que está sendo alterado/
- Background-color, está dando a cor de fundo desta div, em #ffdddd que é um rosa maus claro;
- Novamente, border está definindo a borda da div colorido de vermelho num estilo de borda “pontilhado”;
- Margin, é o espaçamento que as divs tem entre elas, nesse caso é 50px de distância;
- Width é a lagura, que agora equivale a 400px;
- Text-Align, é onde o texto está na div, aqui ele está localizado na esquerda, colorido de vermelho.
